# Deep NLP algorithms based on PyTorch and AllenNLP

## Requirements
 - Python 3.6 (Anaconda preferred)
 - PyTorch 0.4.1
 - AllenNLP latest (`pip install git+git://github.com/allenai/allennlp.git`)

## Steps to reproduce
 1. python train.py
 2. python evaluate.py
 3. python predict.py
 4. ./webdemo.sh
